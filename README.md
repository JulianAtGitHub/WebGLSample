# WebGLSample
WebGl Example Of PBR

## Steps to run this example:
Install NodeJS.

Install gulp-cli globally
>

    npm -g install gulp-cli

Goto root folder and install pacjage
>

    cd path/to/root/folder
    npm i

Build
>

    gulp default

Start a web-server
>

    gulp run-server

Open web browser and access http://localhost:8001
